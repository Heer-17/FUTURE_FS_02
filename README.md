## 📋 Task Overview
Built a Mini CRM (Client Relationship Management) system to manage client leads generated from website contact forms.

## 🔐 Admin Login

## ✅ Features
- Secure Admin Login system
- Dashboard with live stats (Total, New, Contacted, Converted)
- Add / Edit / Delete Leads
- Lead status tracking (New → Contacted → Converted)
- Notes & follow-ups per lead
- Search leads by name/email/company
- Filter leads by status
- Lead source analytics chart
- Lead detail modal popup
- Data saved in LocalStorage (persists on refresh)
- Fully responsive (mobile + tablet + desktop)
